 # 2021 年 APMCM 亚太地区大学生数学建模竞赛参赛指南

 ## 概览

 欢迎参加 2021 年第 11 届 APMCM 亚太地区大学生数学建模竞赛！本指南提供了参赛说明和论文提交要求。了解这些规定对于确保竞赛的公平性和有效性至关重要。

 ## 主要内容

 ### 1. 参赛要求

 **摘要页和论文要求**：

 - 摘要页和论文正文**不得包含参赛者个人信息**，例如姓名、学校或学号。
 - 如发现个人信息，参赛者将被取消评奖资格并仅获得“成功参赛奖”。

 ### 2. 论文提交

 **提交平台**：

 - 参赛者应访问赛氪网站 (www.saikr.com) 提交论文。
 - 登录后，在个人中心按照提示上传论文。

 ### 注意事项

 - 参赛者必须仔细遵守以上规定，以确保论文符合要求。
 - 违规行为将导致论文取消评奖资格。

 ### 竞赛策略

 **团队合作**：

 - 组建由不同背景和技能的成员组成的团队，以带来多角度的观点。
 - 分配清晰的角色和责任，确保团队高效合作。

 **研究和创新**：

 - 充分研究问题并探索多种解决方案。
 - 不要害怕提出创新和非传统的方法。
 - 引用可靠的来源并提供证据来支持您的结论。

 **论文准备**：

 - 遵循竞赛论文格式和提交要求。
 - 使用清晰简练的语言，展示您的研究结果和建模过程。
 - 仔细校对论文以避免语法和拼写错误。
